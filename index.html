Regra de Ouro: Hero Estático

O Hero NUNCA deve ter animações de entrada.

❌ Proibido no Hero

  • • Fade-in, reveal
  • • opacity: 0 → 1
  • • Typing animation

✓ Permitido no Hero

  • • Hover effects
  • • Backgrounds Three.js
  • • Cursor piscando

O Hero precisa renderizar instantaneamente para performance (LCP).

Otimizando para Mobile

Mais da metade dos acessos vem do celular.

Testando no Celular Real

# Descobrir seu IP local

Pedir ao Antigravity

# Acessar no celular

http://192.168.x.x:8888

Requisitos

  • • Celular na mesma rede Wi-Fi
  • • Servidor local rodando
  • • Usar o IP, não localhost

Problemas Comuns no Mobile

Texto muito pequeno

→ Aumentar font-size

Botão difícil de clicar

→ Aumentar padding

Imagem cortada

→ Ajustar aspect-ratio

Scroll horizontal

→ max-width: 100%
